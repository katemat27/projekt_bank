Описание проекта 

Из «Бета-Банка» стали уходить клиенты. 
Каждый месяц. 
Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.

Задача:
Нужно спрогнозировать, уйдёт клиент из банка в ближайшее время или нет. 

Строила модель с предельно большим значением F1-меры. 
Далее была проведена проверка F1-меры на тестовой выборке.
Дополнительно измерялось AUC-ROC и  значение  AUC-ROC сравнивалось с F1-мерой.

Были предоставлены исторические данные о поведении клиентов и расторжении договоров с банком.
Источник данных: https://www.kaggle.com/barelydedicated/bank-customer-churn-modeling
